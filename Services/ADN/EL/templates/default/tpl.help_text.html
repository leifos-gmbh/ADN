E-Learning Hilfe

Hinweis: Für die Funktionen im E-Learning-Bereich sind keine Anmeldung und keine Berechtigung notwendig. Es werden keine personalisierten Funktionen angeboten oder Daten gespeichert.

E-Learning-Einstiegsseite

Auf der Einstiegsseite wird die Liste der Prüfungsbereiche mit der Möglichkeit der Auswahl eines jeweiligen Prüfungsbereichs angezeigt. Der Aktionslink "ADN Test starten" gilt für den jeweils ausgewählten Prüfungsbereich.

Individuellen Test erstellen

Diese Funktion steht Ihnen auf der Einstiegsseite zur Verfügung.
  1. Wählen Sie einen Prüfungsbereich durch Klicken auf das jeweilige Optionsfeld aus.
  2. Klicken Sie auf "ADN Test starten".
  3. Das ADN-System erstellt in Abhängigkeit des Prüfbereiches einen Online-Test entsprechend des MC-Prüfungsteils einer Prüfung.
  4. Das ADN-System zeigt die Fragenübersicht an.

Fragenübersicht

Diese Funktion steht Ihnen nach der Generierung des Online-Tests zur Verfügung.

Das ADN-System zeigt die generierten Fragen, dargestellt als Spalte "Frage" mit einer vorangestellten Fragennummer in einer Übersicht an. Die folgende Spalte mit der Überschrift "Beantwortet" hat initial für jede Frage den Wert "nein". Erst nach Beantwortung der jeweiligen Frage ändert sich der Wert bei erneutem Aufruf der Liste auf "ja". Eine weitere Tabellenspalte der Fragen-Übersichtsdarstellung beinhaltet für jede Frage initial die Aktion "Frage beantworten". Nach Beantwortung der Frage zeigt das System in der Übersichtsdarstellung stattdessen die Aktion "Frage erneut beantworten" mit der Möglichkeit der Ausführung an.

Weiter- oder zurückblättern bzw. zurück zur Fragenübersicht

Diese Funktion steht Ihnen auf der Seite einer einzelnen Frage zur Verfügung.
  1. Wählen Sie eine Antwort aus, bestätigen Sie diese durch Betätigen des Buttons "Antwort bestätigen" und klicken Sie auf "Nächste Frage" bzw. "Vorige Frage".
  2. Das ADN-System speichert die Antwort und zeigt die nächste bzw. vorige Frage an.
  3. Nach Betätigung des Buttons "zurück zur Fragenübersicht" gelangt man zurück zur Fragenübersicht.
Hinweis: Blättern Sie ohne Antwort weiter oder zurück, speichert das ADN-System zwar keine Daten, zeigt die nächste bzw. vorige Frage an und gibt eine Statusmeldung aus "Sind Sie sicher, dass Sie fortfahren möchten, ohne diese Frage zu beantworten?" Mit Wahl von "OK" bzw. "Abbrechen" kann der Dialog beendet werden.

E-Learning-Test beenden

Diese Funktion steht Ihnen auf der Seite der Fragenübersicht zur Verfügung.
  1. Möchten Sie den Test beenden, wechseln Sie bitte in die Übersichtsdarstellung der Fragen und betätigen Sie den Button "Test beenden und Ergebnis anzeigen".
  2. Das ADN-System zeigt eine Sicherheitsabfrage "Möchten Sie den Test wirklich beenden?" und zwei Aktionsknöpfe "Test beenden und Ergebnis anzeigen" und "zurück zur Fragenübersicht" an.
  3. Klicken Sie auf "Test beenden und Ergebnis anzeigen".
  4. Das ADN-System stellt in der Fragenübersicht die richtig beantworteten Fragen mit grüner Hintergrundfarbe sowie die falsch beantworteten Fragen mit roter Hintergrundfarbe dar.
  5. Des weiteren erscheint oberhalb der Tabelle der Fragen das Testergebnis mit der Meldung wie viele Fragen richtig beantwortet wurden und wie viele Punkte erzielt wurden.
  6. Ein Aktionslink "Testergebnis als PDF downloaden" bietet die Möglichkeit, das Testergebnis in einer Datei zur Verfügung zu stellen.
  7. Der Aktionslink "Neuen ADN Test starten" führt zum Beginn eines ADN Tests.

Lösungsbogen herunterladen

Diese Funktion steht Ihnen auf der Seite der Fragenübersicht zur Verfügung.
  1. Klicken Sie auf "Testergebnis als PDF downloaden".
  2. Das ADN-System zeigt den Report E-Learning Lösungsbogen an: Der Report listet alle Fragen mit gegebener und richtiger Antwort auf. Je Frage:
    • Fragentext
    • Gegebene (Ihre) Antwort
    • Richtige Antwort
  3. Laden Sie den Report E-Learning Lösungsbogen herunter (siehe Abschnitt "Dateien herunterladen").

Dateien herunterladen

Bitte vergewissern Sie sich vor der Nutzung, welchen Browser Sie verwenden und gehen Sie im Weiteren anhand des für Sie passenden Schemas vor. Wir beschreiben hier zwei Anwendungsszenarien, nämlich die Nutzung von Internet Explorer und von Mozilla Firefox, beide unter dem Betriebssystem Windows.

Internet Explorer

Nachdem Sie auf "... downloaden" geklickt haben, erscheint ein Dialogfeld "Dateidownload". Sie können nun auf zwei verschiedene Weisen vorgehen:
  1. Sie können die Datei abspeichern. Daraufhin öffnet sich ein Windows-Dialogfeld, wo Sie u. a. den genauen Speicherort angeben können. Nach dem Abspeichern können Sie die Datei dann über ein weiteres Dialogfeld öffnen.
  2. Sie können die Datei sofort öffnen. Es öffnet sich das Standardprogramm für die Anzeige von PDF-Dateien (z. B. Adobe Reader).

Mozilla Firefox

Nachdem Sie auf "... downloaden" geklickt haben, erscheint ein Dialogfeld "Öffnen von ...". Sie können nun auf zwei verschiedene Weisen vorgehen:
  1. Sie können die Datei abspeichern. Es öffnet sich der Download-Manager von Mozilla Firefox. Durch Doppelklick auf den Titel können Sie die Datei nun öffnen.
  2. Sie können die Datei sofort öffnen. Es öffnet sich das Standardprogramm für die Anzeige von PDF-Dateien (z. B. Adobe Reader).
