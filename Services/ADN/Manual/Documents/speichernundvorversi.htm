Speichern und Vorversion sichern

Vorbedingungen:

•      Das ADN-System zeigt das Formular MC-Frage.

 

Gehen Sie folgendermaßen vor:

1.  Klicken Sie statt „Speichern“ auf „Speichern und Vorversion sichern“.

2.  Das ADN-System speichert zusätzlich zum normalen Speichervorgang die alte Version der Frage als Vorversion ab.

3.  Das ADN-System zeigt die Tabelle MC-Fragen an. Es wird eine Erfolgsmeldung angezeigt: „MC-Frage wurde gespeichert.“

Für die bearbeitete Frage wird nun ein zusätzlicher Aktionslink „Vorversion anzeigen“ angezeigt.

 

 

Fehlerquelle:
Fehler A1: Mit leerem Pflichtfeld abspeichern
Fehler E1: Eingabe einer für das Prüfungsziel/Unterziel bereits existierenden Fragennummer

 

 

Hinweis:
Pro Frage wird von dem ADN-System nur eine Vorversion gepflegt.
Bei diesem Vorgang wird daher eine evtl. vorher vorhandene Vorversion überschrieben.
