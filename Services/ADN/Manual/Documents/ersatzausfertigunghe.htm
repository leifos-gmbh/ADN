Ersatzausfertigung herunterladen

Vorbedingungen:

•      Das ADN-System zeigt die Tabelle ADN-Bescheinigungen an.

•      Für mindestens eine ADN-Bescheinigung liegt bereits eine Ersatzausfertigung vor.

 

Gehen Sie folgendermaßen vor:

1.  Klicken Sie auf der Aktionslink „Verlängerung herunterladen“ eines Sachverständigen.

Das ADN-System bietet den Report Ersatzausfertigung als PDF-Dokument zum Herunterladen an. In die Vorlage sind die gleichen Daten wie im Report ADN-Bescheinigung individualisiert eingefügt. Zusätzlich wird der Text „Ersatzausfertigung“ in das Dokument als Überschrift eingefügt.

2.  Laden Sie den Report Ersatzausfertigung herunter (siehe Kapitel 2.1 Herunterladen von Dateien).
