Fachgebiet bearbeiten

Vorbedingungen:

•      Das ADN-System zeigt die Tabelle Fachgebiete an.

•      Mindestens ein Fachgebiet ist angelegt.

 

Gehen Sie folgendermaßen vor:

1.  Klicken Sie auf „Bearbeiten“ eines Fachgebietes.

2.  Das ADN-System zeigt das ausgefüllte Formular Fachgebiet an.

3.  Ändern Sie die Daten so, dass sie korrekt und vollständig bleiben.

4.  Klicken Sie auf „Speichern“.

Das ADN-System zeigt die Tabelle Fachgebiete an.

Es wird eine Erfolgsmeldung angezeigt: „Fachgebiet wurde gespeichert“.

 

 

Fehlerquelle:
Fehler A1: Mit leerem Pflichtfeld abspeichern
