Schulungsorte löschen

Vorbedingungen:

•      Das ADN-System zeigt die Tabelle Schulungsorte an.

 

Gehen Sie folgendermaßen vor:

1.  Markieren Sie einen oder mehrere Schulungsorte mit Hilfe der Auswahlboxen.

2.  Klicken Sie auf „Löschen“.

3.  Das ADN-System listet die Objekte und fragt „Wollen Sie folgende Schulungsorte wirklich löschen?“ angezeigt. Es werden zwei Aktionsknöpfe „Löschen“ und „Abbrechen“ angezeigt.

4.  Klicken Sie auf „Löschen“.

Das ADN-System löscht die markierten Schulungsorte.

Das ADN-System kehrt zurück zur Tabelle Schulungsorte.

Es wird eine Erfolgsmeldung angezeigt: „Schulungsort wurde gelöscht“.

 

 

Fehlerquelle:
Fehler L1: Ohne Auswahl löschen

 

 

Hinweis:
Wird ein Schulungsort gelöscht, der einem bereits archiviertem Schulungstermin zugewiesen ist, so wird er darin weiterhin angezeigt, kann aber in neuen Terminen nicht mehr neu zugeordnet werden.
