Übersicht über Prüfungskandidaten/Sachkundige anzeigen

Gehen Sie folgendermaßen vor:

1.  Wählen Sie den Menüpunkt „Prüfungsvorbereitung | Prüfungskandidaten“ aus.

2.  Das ADN-System zeigt eine Auswahlbox „Sachkundige einblenden“ an.

3.  Das ADN-System zeigt eine Auswahlbox „Interessenten einblenden“ an.

4.  Das ADN-System zeigt den Aktionsknopf „Ansicht aktualisieren“ an.

5.  Das ADN-System zeigt den Aktionsknopf „Kandidat hinzufügen“ an.

6.  Das ADN-System zeigt die Tabelle Prüfungskandidaten/Sachkundige an:

Abbildung 56: Tabelle Prüfungskandidat

Das System erlaubt eine alphabetische bzw. numerische Sortierung des Tabelleninhaltes aller Inhaltsspalten in aufsteigender oder absteigender Reihenfolge. Es werden insgesamt sieben Inhaltsspalten angezeigt:

•      Name

•      Vorname

•      Geburtsdatum

•      Staatsangehörigkeit

•      Zu absolvierende Prüfung

•      Erfasst durch

•      Sperrfrist bis (Datum, wenn gegeben)

 

Die Spalte „Aktionen“ zeigt je Prüfungskandidat/Sachkundigem einen Link „Kandidat bearbeiten“.
