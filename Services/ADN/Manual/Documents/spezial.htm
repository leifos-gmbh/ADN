Spezial

Fehler S1       Einladungen generieren ohne zugeordneten Kandidaten

Gehen Sie folgendermaßen vor:

1.  Ordnen Sie einen Kandidaten zu.

2.  Klicken Sie auf „Einladungen generieren“ zu einem Prüfungstermin.

 

Fehler S2       Einladungen generieren ohne Auswahl

Gehen Sie folgendermaßen vor:

1.  Wählen Sie eine Auswahlbox und klicken Sie auf „Einladungen generieren“.

2.  Das ADN-System zeigt die Tabelle „Einladungen“ erneut mit einer Fehlermeldung: „Bitte wählen Sie mindestens ein Objekt aus“ an.

 

Fehler S3       Ohne Auswahl fortsetzen

Gehen Sie folgendermaßen vor:

1.  Klicken Sie auf „Fragen hinzufügen“ zu einem Ziel/Unterziel.

2.  Das ADN-System zeigt eine Liste von Fragen mit jeweils einer Auswahlbox und einem Aktionsknopf „Fragen hinzufügen“. In der Liste erscheinen Fragen.

3.  Aktivieren Sie eine Auswahlbox.

4.  Klicken Sie „Fragen hinzufügen“.

 

Fehler S4       Mehr Fragen auswählen als dem Ziel noch zugeordnet werden können

Gehen Sie folgendermaßen vor:

1.  Klicken Sie auf „Fragen hinzufügen“ eines Ziel/Unterziel.

2.  Das ADN-System zeigt eine Liste von Fragen mit jeweils einer Auswahlbox und einem Aktionsknopf „Fragen hinzufügen“. In der Liste erscheinen Fragen.

3.  Aktivieren Sie die korrekte Anzahl an Auswahlboxen.

4.  Klicken Sie auf „Fragen hinzufügen“.

 

Fehler S5       Ohne Teilnehmerauswahl fortsetzen

Gehen Sie folgendermaßen vor:

1.  Aktivieren Sie keine Auswahlbox eines Prüfungsteilnehmers

2.  Klicken Sie auf den Aktionsknopf „Antwortschreiben erstellen“.
