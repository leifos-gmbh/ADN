Zulassungszeugnis bearbeiten

Vorbedingungen:

•      Das ADN-System zeigt die Tabelle Zulassungszeugnisse (Chemie) oder die Tabelle Zulassungszeugnisse (Gas) an.

 

Gehen Sie folgendermaßen vor:

1.  Klicken Sie auf “Bearbeiten“ eines Zulassungszeugnisses.

2.  Das ADN-System zeigt das Formular Zulassungszeugnis mit den Daten des ausgewählten Zulassungszeugnisses an.

3.  Ändern Sie die Daten so, dass sie korrekt und vollständig bleiben. Ist eine Datei bereits vorhanden, so kann sie durch aktivieren der Auswahlbox „Bestehende Datei löschen“ entfernt werden.

4.  Klicken Sie auf „Speichern“.

5.  Das ADN-System zeigt die Tabelle Zulassungszeugnisse (Chemie) an.

Es wird die Erfolgsmeldung: „Zulassungszeugnis wurde gespeichert.“ angezeigt.

 

 

Fehlerquelle:
Fehler A1 Mit leerem Pflichtfeld abspeichern
