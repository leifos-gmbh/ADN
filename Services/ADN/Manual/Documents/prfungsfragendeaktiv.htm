Prüfungsfragen deaktivieren

Vorbedingungen:

•      Das ADN-System zeigt die Tabelle MC-Fragen oder die Tabelle Kasus-Fragen an.

•      Es besteht mindestens eine aktive Frage.

 

Gehen Sie folgendermaßen vor:

1.  Markieren Sie über die Auswahlboxen eine oder mehrere Fragen.

2.  Suchen Sie aus der Auswahlliste die Funktion „Deaktivieren“.

3.  Klicken Sie auf „Ausführen“.

4.  Das ADN-System setzt den Status der ausgewählten Fragen auf inaktiv und speichert das Datum der Deaktivierung,

•      Das ADN-System zeigt die Tabelle MC-Fragen bzw. die Tabelle Kasus-Fragen an.

Es wird eine Erfolgsmeldung angezeigt: „Fragen wurden deaktiviert“.

 

 

Fehlerquelle:
Fehler AK1: Ohne Auswahl de-/aktivieren
