Übersicht der Sachkundigen/Bescheinigungen anzeigen

Vorbedingungen:

•      Es ist mindestens ein Sachkundiger gespeichert, d. h., mindestens eine Person hat bereits eine Prüfung abgelegt und bestanden.

 

Gehen Sie folgendermaßen vor:

1.  Wählen Sie den Menüpunkt „Sachkundigenverwaltung | Sachkundige/Bescheinigungen“.

2.  Das ADN-System zeigt die Tabelle ADN-Bescheinigungen an:

Abbildung 81: Tabelle ADN-Bescheinigungen

Die Tabelle besitzt fünf Inhaltsspalten mit den Eigenschaften des Objektes:

•      Nummer der Bescheinigung

•      Name

•      Vorname

•      Geburtsdatum

•      Gültig bis

 

Die Spalte „Aktionen“ weist je Bescheinigung eines Sachkundigen maximal folgende Aktionslinks auf:

•      Details anzeigen

•      Verlängerung erstellen

•      Bearbeiten

•      Kostenbescheid erstellen         (falls die Bescheinigung gültig ist)

•      Kostenbescheid herunterladen (falls die Bescheinigung gültig ist)

•      Archivierte Bescheinigungen     (falls zu diesem Sachkundigen weitere anzeigen archivierte Bescheinigungen vorliegen)

•      Verlängerung herunterladen     (falls eine Verlängerung erstellt wurde)

•      Ersatzausfertigung erstellen     (falls die Bescheinigung gültig ist und der Benutzer bei der ausstellenden WSD arbeitet)

•      Ersatzausfertigung herunterladen (falls eine Ersatzausfertigung erstellt)

 

Das System erlaubt eine alphabetische bzw. numerische Sortierung des Tabelleninhaltes jeder dafür vorgesehenen Spalte in aufsteigender oder absteigender Reihenfolge.

Oberhalb der Tabelle befindet sich eine Auswahlbox, um „Archivierte Bescheinigungen“ einzublenden. Hinter der Auswahlbox befindet sich der Aktionsknopf „Ansicht aktualisieren“.

 

 

Hinweis:
Wird für eine Bescheinigung eine Verlängerung erstellt oder erhält ein Sachkundiger aus anderen Gründen eine neue ADN-Bescheinigung, so bleibt die frühere Version in einem Archiv gespeichert, die aber nicht mehr gültig ist. Archivierte Bescheinigungen können insgesamt in die Übersicht eingeblendet oder für einen einzelnen Sachkundigen eingesehen werden.
Mit der Zeit können für einen Sachkundigen so mehrere archivierte Versionen entstehen.
