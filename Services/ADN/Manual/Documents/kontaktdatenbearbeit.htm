Kontaktdaten bearbeiten

Vorbedingungen:

•      Das ADN-System zeigt die Tabelle Schulungsveranstalter an.

 

Gehen Sie folgendermaßen vor:

1.  Klicken Sie auf den Aktionslink „Kontaktdaten bearbeiten“ eines Schulungsveranstalters.

2.  Das ADN-System zeigt vier Reiter an:

•      Kontaktdaten (aktiviert)

•      Anerkannte Kursarten

•      Dozenten

•      Schulungsorte

Darunter zeigt das ADN-System das ausgefüllte Formular Schulungsveranstalter mit den Daten des ausgewählten Schulungsveranstalters an.

3.  Ändern Sie die Daten so, dass sie korrekt und vollständig bleiben.

4.  Klicken Sie auf „Speichern“.

5.  Das ADN-System zeigt die Tabelle Schulungsveranstalter an.

Es wird eine Erfolgsmeldung angezeigt: „Schulungsveranstalter wurde gespeichert“.

 

 

Fehlerquelle:
Fehler A1 Mit leerem Pflichtfeld abspeichern
