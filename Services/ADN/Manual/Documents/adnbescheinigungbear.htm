ADN-Bescheinigung bearbeiten

Vorbedingungen:

•      Das ADN-System zeigt die Tabelle Prüfungskandidaten/ADN-Bescheinigungen.

•      Es gibt Prüfungsteilnehmer, für die bereits eine ADN-Bescheinigung ausgestellt wurde.

 

Gehen Sie folgendermaßen vor:

1.  Klicken Sie auf den Aktionslink „Bescheinigung bearbeiten“ einer Person.

2.  Das ADN-System zeigt das Formular ADN-Bescheinigung an.

Zusätzlich wird oberhalb der bearbeitbaren Felder die Nummer der Bescheinigung angezeigt (Fester Wert: Behördenkennziffer - fortlaufende Nummer (dreistellig, beginnend mit 1 pro Jahr, ausgelassene Nummern werden aufgefüllt)-Jahreszahl, Beispiel „7-036-2010“).

3.  Ändern Sie die Daten so, dass sie korrekt und vollständig bleiben.

4.  Klicken Sie auf „Speichern“.

5.  Das ADN-System zeigt die Tabelle Prüfungstermine/ADN-Bescheinigungen an.

Es wird eine Erfolgsmeldung angezeigt: „Bescheinigung wurde erstellt“.

 

 

Fehlerquelle:
Fehler A1: Mit leerem Pflichtfeld abspeichern

 

 

Hinweis:
Eine evtl. vorher vorhandene Bescheinigung wird in diesem Moment archiviert. Personen haben immer nur eine (oder keine) gültige Bescheinigung. Bescheinigungen, welche aufgrund einer Prüfung ausgestellt wurden, bekommen immer eine neue Nummer, selbst wenn die Person vorher eine Bescheinigung hatte. Das System muss sich intern merken, dass eine Bescheinigung zu einer bestimmten Prüfung erstellt wurde, um deren Bearbeitung zu ermöglichen ohne eine weitere Bescheinigung aufgrund derselben Prüfung auszustellen. Wenn alle Eingaben gültig sind, werden die Daten gespeichert.
