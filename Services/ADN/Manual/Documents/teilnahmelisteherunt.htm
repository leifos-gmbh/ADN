Teilnahmeliste herunterladen

Vorbedingungen:

•      Das ADN-System zeigt die Tabelle Prüfungstermine/Teilnahmeliste an.

•      Für mindestens einen Prüfungstermin wurde eine Teilnahmeliste erstellt.

 

Gehen Sie folgendermaßen vor:

1.  Klicken Sie auf „Teilnahmeliste herunterladen“ zu einem Prüfungstermin an.

2.  Das ADN-System bietet den Report Teilnahmeliste über den Browser zum Herunterladen an (siehe Kapitel 2.1 Herunterladen von Dateien). Die Datei basiert auf der Vorlage Report_Teilnahmeliste_Basis.pdf, inkl. Rechtsbehelfsbelehrung.

In die Vorlage werden folgende kandidatenspezifischen Daten eingefügt:

•      Art der Prüfung

•      Datum der Prüfung

•      Ort der Prüfung

•      Anrede + Name des Vorsitzenden

•      Anrede + Name des Beisitzer 1

•      Anrede + Name des Beisitzer 2

•      Name der WSD

•      Adresse der WSD

•      Rechtsbehelfsbelehrung          (soll immer mit ausgegeben werden, kann auch Teil der Vorlage werden)

•      Für jeden Prüfungsteilnehmer

•      Nr.

•      Name

•      Vorname

•      Geburtsdatum

•      Staatsangehörigkeit
