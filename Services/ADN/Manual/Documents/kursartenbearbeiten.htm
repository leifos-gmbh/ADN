Kursarten bearbeiten

Vorbedingungen:

•      Das ADN-System zeigt die Tabelle Schulungsveranstalter an.

 

Gehen Sie folgendermaßen vor:

1.  Klicken Sie auf den Aktionslink „Kursarten bearbeiten“ eines Schulungsveranstalters.

2.  Über dem Formular werden vier Reiter angezeigt:

•      Kontaktdaten

•      Anerkannte Kursarten           (aktiviert)

•      Dozenten

•      Schulungsorte

 

Das ADN-System zeigt das Formular Anerkannte Kursarten an:

Abbildung 12: Formular Anerkannte Kursarten

 

Das ADN-System zeigt eine Tabelle aller anerkannten Kursarten an:

•      Basiskurs Trockengüterschifffahrt

•      Basiskurs Tankschifffahrt

•      Basiskurs kombiniert Trockengüter-/Tankschifffahrt

•      Aufbaukurs Gas

•      Aufbaukurs Chemie

•      Wiederholungskurs Trockengüterschifffahrt

•      Wiederholungskurs Tankschifffahrt

•      Wiederholungskurs kombiniert Trockengüter-/Tankschifffahrt

•      Wiederholungskurs Gas

•      Wiederholungskurs Chemie

 

Hinter jeder Kursart wird ein Aktionsfeld angezeigt. Ist dieses aktiviert, wird das Datum der Genehmigung angezeigt.

3.  Aktivieren Sie ein Aktionsfeld und/oder stellen Sie das Datum der Genehmigung (neu) ein.

4.  Klicken Sie auf „Speichern“.

5.  Das ADN-System zeigt das Formular Schulungsveranstalter an.

Es wird eine Erfolgsmeldung angezeigt: „Anerkannte Kursarten wurden gespeichert“.

 

 

Hinweis:

Sie können das Formular auch über Reiter in anderen Ansichten der Schulungsveranstalter aufrufen.
