Stoffabhängige Antwort bearbeiten

Vorbedingungen:

•       Das ADN-System zeigt die Tabelle Stoffabhängige Antworten an.

•       Es besteht mindestens eine stoffabhängige Antwort.

 

Gehen Sie folgendermaßen vor:

1.  Klicken Sie auf „Bearbeiten“ zu einer Antwort.

2.  Das ADN-System zeigt das Formular Stoffabhängige Antwort mit den Daten des ausgewählten Schulungsveranstalters an. Über dem Formular werden vier Reiter angezeigt:

•      Antwort   (Text, CLOB, Sonderzeichen zugelassen)

•      Butan oder Leer (Auswahlmenü, nur im Bereich Gas, Vorauswahl: Leer)

Für jeden Stoff aus dem entsprechenden Bereich (Gas/Chemie) wird eine Auswahlbox angeboten.

3.  Ändern Sie die Daten so, dass sie korrekt und vollständig bleiben.

4.  Klicken Sie auf „Speichern“.

5.  Das ADN-System zeigt die Tabelle Stoffabhängige Antworten an.

Es wird eine Erfolgsmeldung angezeigt: „Stoffabhängige Antwort wurde gespeichert“.

 

 

Fehlerquelle:
Fehler A1 Mit leerem Pflichtfeld abspeichern.
