MC-Prüfungsfrage bearbeiten

Vorbedingungen:

•      Das ADN-System zeigt die Tabelle MC-Fragen an.

•      Es existiert mindestens eine andere MC-Frage. Diese hat den Status „inaktiv“ (siehe Kapitel 5.3.3.1 Prüfungsfrage aktivieren).

 

Gehen Sie folgendermaßen vor:

1.  Klicken Sie auf „Bearbeiten“ für eine inaktive Frage.

2.  Das ADN-System zeigt das Formular MC-Frage.

3.  Ändern Sie die Daten so, dass sie korrekt und vollständig bleiben.

4.  Klicken Sie auf „Speichern“.

5.  Das ADN-System zeigt weiterhin das Formular MC-Frage an.

Es wird die Erfolgsmeldung: „MC-Frage wurde gespeichert“ angezeigt.

 

 

Fehlerquelle:
Fehler A1: Mit leerem Pflichtfeld abspeichern
Fehler E1: Eingabe einer für das Prüfungsziel/Unterziel bereits existierenden Fragennummer

 

 

Hinweis:
Sie können auch eine Fragennummer eintragen, die innerhalb des Prüfungsziels/Unterziels, dem Sie die Frage zuordnen, schon verwendet wurde, wenn diese bereits wieder gelöscht ist.
In der archivierten Frage wird die Fragennummer trotzdem weiterhin verwendet.
