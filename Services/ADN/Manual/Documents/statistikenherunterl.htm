Statistiken herunterladen

Vorbedingungen:

•      Es wird eine der Sichten aus dem Menü „Statistik“ angezeigt.

 

Gehen Sie folgendermaßen vor:

1.  Klicken Sie auf der Aktionslink „Statistik herunterladen“ oberhalb der Tabelle.

2.  Das ADN-System bietet je nach Statistikansicht verschiedene Reports als PDF zum Herunterladen an:

•      Report Durchgeführte Prüfungen

•      Report Verlängerung aufgrund Wiederholungskurs

•      Report Verlängerung aufgrund Arbeitszeitnachweis

•      Report Bescheinigung aufgrund sonstiger Anträge

•      Report Bescheinigungen für Aufbaukurse Gas und Chemie

•      Report Summe Bescheinigungen

Die Reporte enthalten jeweils den Inhalt der Tabelle.

Darüber wird eingetragen:

•      „Ausbildung von Sachkundigen nach Kapitel 8.2 ADN“

•      Art der Statistik

•      Statistik für den Zeitraum vom [n] bis zum [n]

3.  Laden Sie den jeweiligen Report herunter (siehe Kapitel 2.1 Herunterladen von Dateien).
