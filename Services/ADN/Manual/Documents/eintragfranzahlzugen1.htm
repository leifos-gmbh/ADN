Eintrag für Anzahl zu generierender Prüfungsfragen bearbeiten

Beispiel Trockengüterschifffahrt

Vorbedingungen:

•      Das ADN-System zeigt die Tabelle Anzahl zu generierender Fragen im Register Trockengüterschifffahrt an.

 

Gehen Sie folgendermaßen vor:

1.  Klicken Sie auf „Bearbeiten“ zu einem Eintrag.

2.  Das ADN-System zeigt das Formular Eintrag Anzahl zu generierender Fragen im Register Trockengüterschifffahrt an.

3.  Ändern Sie die Daten so, dass sie korrekt und vollständig bleiben.

4.  Klicken Sie auf „Speichern“.

5.  Das ADN-System überprüft die Eingaben.

6.  Das ADN-System zeigt die Tabelle Anzahl zu generierender Fragen im Register Trockengüterschifffahrt an. Es wird eine Erfolgsmeldung angezeigt: „Eintrag wurde gespeichert“.

 

 

Fehlerquelle:
Fehler A1: Mit leerem Pflichtfeld abspeichern

Hinweis:

Bitte beachten Sie, dass auch beim nachträglichen Bearbeiten des Eintrags die Möglichkeit besteht, ein Maximum von einer Frage je Prüfungsziel (bzw. Unterziel) einzustellen (siehe Kapitel 5.2.1.4 Maximum von einer Frage je Ziel einstellen)
