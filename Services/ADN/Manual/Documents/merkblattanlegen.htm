Merkblatt anlegen

Vorbedingungen:

•      Das ADN-System zeigt die Tabelle Merkblätter an.

 

Gehen Sie folgendermaßen vor:

1.  Klicken Sie auf „Merkblatt hinzufügen“.

2.  Das ADN-System zeigt das Formular Merkblatt an:

Abbildung 22: Formular Merkblatt

Am Ende der Seite befinden sich die Aktionsknöpfe „Speichern“ und „Abbrechen“. Im Formular werden Pflichteingaben mit einem Stern „*“ gekennzeichnet. Das ADN-System zeigt ein Formular mit folgenden Eingabefeldern:

•      Name      (Text, Länge 200, Pflichtfeld)

•      Datei       (Dateiupload, Pflichtfeld mit dem Aktionsknopf „Durchsuchen…“)

3.  Tragen Sie einen Namen ein und wählen Sie eine Datei von Ihrem lokalen Rechner aus.

4.  Klicken Sie auf „Speichern“.

Das ADN-System zeigt die Tabelle Merkblätter an.

Es wird eine Erfolgsmeldung angezeigt: „Merkblatt wurde hinzugefügt“.

 

 

Fehlerquelle:
Fehler A1: Mit leerem Pflichtfeld abspeichern
