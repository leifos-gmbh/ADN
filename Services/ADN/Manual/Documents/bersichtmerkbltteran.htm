Übersicht Merkblätter anzeigen

Gehen Sie folgendermaßen vor:

1.  Wählen Sie den Menüpunkt „Schulungsverwaltung | Merkblätter“ aus.

Das ADN-System zeigt den Aktionsknopf „Merkblätter hinzufügen“ an.

Das ADN-System zeigt die Tabelle Merkblätter an:

Abbildung 21: Tabelle Merkblätter

Über und unter der Tabelle befinden sich sie Schaltflächen „Löschen“.

In der ersten Spalte wird eine Aktionsbox angezeigt.

Das System erlaubt eine alphabetische Sortierung des Tabelleninhaltes der ersten Spalte in aufsteigender oder absteigender Reihenfolge.

Das ADN-System zeigt eine Inhaltsspalte an:

•      Name

Je Merkblatt zeigt das ADN-System einen Aktionslink an:

•      Herunterladen
