Situationsbeschreibungen bearbeiten

Vorbedingungen:

•      Das ADN-System zeigt das Formular Situationsbeschreibung  an.

 

Gehen Sie folgendermaßen vor:

1.  Klicken Sie einen der Reiter

•      Gas (leer)

•      Gas (Butan)

•      Chemie

2.  Tragen Sie einen Text ein.

Platzieren Sie dabei im Text ggf. einen oder mehrere der jeweils zugelassenen Platzhalter.

3.  Klicken Sie auf „Speichern“.

4.  Das ADN-System zeigt die Erfolgsmeldung: „Situationsbeschreibung wurde gespeichert.“

Das Formular wird weiterhin angezeigt.

 

 

Fehlerquelle:
Fehler A1 Mit leerem Pflichtfeld abspeichern
