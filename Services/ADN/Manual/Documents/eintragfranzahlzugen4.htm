Eintrag für Anzahl zu generierender MC-Fragen bearbeiten

Beispiel Chemie

 

Vorbedingungen:

•      Das ADN-System zeigt das Formular Eintrag Anzahl zu generierender Fragen –Aufbau im Register „Aufbau Chemie“ an.

 

Gehen Sie folgendermaßen vor:

1.  Klicken Sie auf „Bearbeiten“.

2.  Das ADN-System zeigt das Formular Eintrag Anzahl zu generierender Fragen –Aufbau-Kasus mit folgenden Eingabefeldern:

•      Anzahl                        (Numerisch, Länge 2, Pflichtfeld)

•      Max. eine Frage je Ziel (Auswahlbox)

Für jedes Prüfungsziel/Unterziel wird eine Auswahlbox angezeigt.

3.  Ändern Sie die Daten so, dass sie korrekt und vollständig bleiben.

4.  Klicken Sie auf „Speichern“.

5.  Das ADN-System zeigt die Tabelle Eintrag Anzahl zu generierender Fragen–Aufbau an.

Es wird eine Erfolgsmeldung angezeigt: „Eintrag wurde gespeichert“.

 

 

Fehlerquelle:
Fehler A1: Mit leerem Pflichtfeld abspeichern
