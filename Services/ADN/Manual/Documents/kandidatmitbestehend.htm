Kandidat mit bestehender Sperrfrist anmelden

 

Hinweis:
Fällt ein Kandidat bei einer Prüfung durch, so wird er in der Regel mit einer Sperrfrist versehen, während derer er an keiner erneuten Prüfung teilnehmen darf. Soll nun ein Prüfungskandidat für eine Prüfung abgespeichert werden, obwohl für ihn noch eine Sperrfrist gilt, so kann dies angegeben werden, indem Sie eine entsprechende Warnmeldung übergehen („Antrag annehmen“).

Wird bei der Warnmeldung „Antrag ablehnen“  angeklickt, so wird beim Datensatz des Prüfungskandidaten im Feld „Antrag auf Prüfung gestellt“ das Häkchen automatisch entfernt.

 

 

Vorbedingungen:

•      Für mindestens einen Kandidaten liegt eine Sperrfrist vor, welche aktuell wirksam ist.

•      Das ADN-System zeigt das Formular Prüfungskandidaten/Sachkundige an.

 

Gehen Sie folgendermaßen vor:

1.  Tragen Sie die Daten korrekt und vollständig ein und markieren Sie insbesondere das Feld „Antrag auf Prüfung gestellt“.

2.  Klicken Sie auf „Speichern“.

3.  Das ADN-System zeigt eine Sicherheitsabfrage „Es liegt eine Sperrfrist zum Prüfungskandidaten vor. Soll der Antrag trotz Sperrfrist gesetzt werden?“ und zwei Aktionsknöpfe „Antrag annehmen“ und „Antrag ablehnen“ an.

4.  Klicken Sie auf „Antrag annehmen“.

5.  Das ADN-System kehrt zur Tabelle Prüfungskandidaten/Sachkundige zurück.

In der Spalte „Zu absolvierende Prüfung“ wird für den Prüfungskandidaten die gewählte Prüfungsart angezeigt.

Das ADN-System zeigt die Erfolgsmeldung an: „Prüfungskandidat wurde gespeichert.“
