Test beenden

Vorbedingungen:

•      Das ADN-System zeigt die Sicht Testfrage mit einer Frage einer Online-Prüfung an.

Gehen Sie folgendermaßen vor:

1.  Wählen Sie eine Antwortmöglichkeit und klicken Sie auf „Test beenden“.

2.  Das ADN-System zeigt eine Sicherheitsabfrage „Wollen Sie den Test wirklich beenden?“ und zwei Aktionsknöpfe „Ja“ und „Nein“ an.

3.  Klicken Sie auf „Ja“.

4.  Das ADN-System speichert die letzte Frage und gibt eine Statusmeldung „Der Test wurde beendet“ aus. Es wird ein Link zum Abmelden vom System angezeigt.

 

 

Hinweis:
Klicken Sie auf „Nein“, kehrt das ADN-System zur letzten ausgewählten Frage zurück.
