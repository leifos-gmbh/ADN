Antwortschreiben herunterladen

Vorbedingungen:

•      Das ADN-System zeigt die Tabelle Prüfungsteilnehmer/Antwortschreiben an.

•      Für einen Teilnehmer ist ein Antwortschreiben generiert worden.

 

Gehen Sie folgendermaßen vor:

1.  Klicken Sie auf den Aktionslink „Antwortschreiben herunterladen“ hinter einem Teilnehmer.

2.  Das ADN-System bietet je nach Situation ein PDF-Dokument nach einer bestimmten Vorlage zum Herunterladen an:

•      Report Antwortschreiben, Nichtbestehen

•      Report Antwortschreiben, Bestehen

3.  Laden Sie den Report herunter (siehe Kapitel 2.1 Herunterladen von Dateien).
