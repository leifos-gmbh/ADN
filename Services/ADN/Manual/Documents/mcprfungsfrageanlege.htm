MC-Prüfungsfrage anlegen

Vorbedingungen:

•      Es wird die Tabelle MC-Fragen angezeigt.

Gehen Sie folgendermaßen vor:

1.  Suche Sie einen Fragenkatalogbereich aus der Auswahlliste aus.

2.  Klicken Sie auf „MC-Frage hinzufügen“.

3.  Das ADN-System zeigt das Formular MC-Frage mit zusätzlichem Aktionsknopf „Speichern“ und den Eingabefeldern:

Abbildung 38: Formular MC-Frage

•      Prüfungsziel/Unterziel: (Auswahlliste aller Prüfungsziele und Unterprüfungsziele des Fragenkatalogbereiches )

•      Nummer (7./.8 Ziffer im Nummernschema, Numerisch, Länge 2, Pflichtfeld)

•      Titel (Text, Länge 200, Pflichtfeld, Sonderzeichen zugelassen (siehe Kapitel 10.4 Sonderzeichen)

•      Wortlaut der Frage (Text, CLOB, Pflichtfeld, Sonderzeichen zugelassen)

•      Hochladen einer Grafikdatei (nur PNG, GIF, JPG-Dateien zugelassen)

•      Status (aktiv/inaktiv)

•      Angabe richtigen Antwort (A, B, C, D)

Abbildung 39: Formular MC-Fragen, Frage

Vier Antworten mit jeweils:

•      Nummerierung A, B, C, D (nur Anzeige)

•      Antworttext (Text, CLOB, Pflichtfeld, Sonderzeichen zugelassen)

•      Eingabefeld zum Hochladen einer Grafikdatei zur Antwort (nur PNG, GIF, JPG-Dateien zugelassen)

4.  Tragen Sie die Daten korrekt und vollständig ein.

5.  Klicken Sie auf „Speichern“.

6.  Das ADN-System zeigt die Tabelle MC-Fragen an. Es wird eine Erfolgsmeldung angezeigt: „Frage wurde angelegt“.

 

 

Fehlerquelle:
Fehler E1: Eingabe einer für das Prüfungsziel/Unterziel bereits existierenden Fragennummer

 

 

Hinweis:
Sie können auch eine Fragennummer eintragen, die innerhalb des Prüfungsziels/Unterziels, dem Sie die Frage zuordnen, schon verwendet wurde, wenn diese bereits wieder gelöscht ist.
In der archivierten Frage wird die Fragennummer trotzdem weiterhin verwendet.
