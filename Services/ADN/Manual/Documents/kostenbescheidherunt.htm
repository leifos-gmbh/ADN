Kostenbescheid herunterladen

Vorbedingungen:

•      Das ADN-System zeigt die Tabelle ADN-Bescheinigungen an.

•      Für mindestens einen Sachkundigen wurde bereits ein Kostenbescheid erstellt.

 

Gehen Sie folgendermaßen vor:

1.  Klicken Sie auf der Aktionslink „Kostenbescheid herunterladen“ einer Bescheinigung eines Sachkundigen.

Das ADN-System bietet den Report Kostenbescheid als PDF-Dokument zum Herunterladen an:

In die Vorlage sind folgende Daten individualisiert eingefügt:

•      Anrede

•      Vorname

•      Name

•      Straße

•      PLZ

•      Ort

•      Name der WSD

•      Adresse der WSD

•      Aus den WSD-Stammdaten:

•      Betrag des jeweiligen Kostengegenstandes

•      Bezeichnungstext des Gegenstandes

•      Laufende Nummer

•      Nummer der Bescheinigung

 

Bezieht sich der Kostenbescheid auf eine Erstausfertigung, so wird dem Betrag der Wert des Datums „Zusätzliche Kosten pro Person“ des Prüfungstermins hinzugerechnet.

Laden Sie den Report Kostenbescheid herunter (siehe Kapitel 2.1 Herunterladen von Dateien).
