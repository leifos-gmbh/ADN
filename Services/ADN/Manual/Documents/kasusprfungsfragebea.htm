Kasus-Prüfungsfrage bearbeiten

Vorbedingungen:

•      Das ADN-System zeigt die Tabelle Kasus-Fragen an.

•      Es existiert mindestens eine Kasus-Prüfungsfrage.

•      Die Frage hat den Status „inaktiv“ (siehe Kapitel 5.3.3.1 Prüfungsfrage aktivieren).

Gehen Sie folgendermaßen vor:

1.  Klicken Sie auf „Bearbeiten“ für eine inaktive Frage.

2.  Das ADN-System zeigt das Formular Kasus-Fragen) (siehe Kapitel 5.3.2.3 Kasus-Prüfungsfrage anlegen).

Es existieren nun zusätzliche Daten und Eingabefelder, die nur beim Bearbeiten einer bereits bestehenden angezeigt werden:

•      Änderungskommentar                               (Text, CLOB)

•      Datum der letzten Änderung                       (nur Anzeige)

•      Benutzer der letzten Änderung                             (nur Anzeige)

•      Kommentar zur letzten Änderung                (nur Anzeige)

•      Datum der letzten Aktivierung/Deaktivierung (nur Anzeige)

3.  Ändern Sie die Daten so, dass sie korrekt und vollständig bleiben.

4.  Klicken Sie auf „Speichern“.

5.  Das ADN-System zeigt die Tabelle Kasus-Fragen an.

Es wird eine Erfolgsmeldung angezeigt: „Kasus-Frage wurde gespeichert.“

 

 

Fehlerquelle:
Fehler A1: Mit leerem Pflichtfeld abspeichern
Fehler E1: Eingabe einer für das Prüfungsziel/Unterziel bereits existierenden Fragennummer

 

 

Hinweis:
Sie können auch eine Fragennummer eintragen, die innerhalb des Prüfungsziels, dem Sie die Frage zuordnen, schon verwendet wurde, wenn diese bereits wieder gelöscht ist.
In der archivierten Frage wird die Fragennummer trotzdem weiterhin verwendet.
