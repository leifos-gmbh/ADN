Kasus-Prüfungsfrage löschen

Vorbedingungen:

•      Das ADN-System zeigt die Tabelle Kasus-Fragen an.

•      Es existiert mindestens eine Kasus-Prüfungsfrage.

Gehen Sie folgendermaßen vor:

1.  Markieren Sie eine oder mehrere Fragen.

2.  Suchen Sie aus der Auswahlliste die Funktion „Löschen“.

3.  Klicken Sie auf „Ausführen“.

4.  Das ADN-System zeigt die Frage „Wollen Sie folgende Kasus-Fragen wirklich löschen?“ Es werden zwei Aktionsknöpfe „Löschen“ und „Abbrechen“ angezeigt.

5.  Klicken Sie auf „Löschen“.

6.  Das ADN-System zeigt die Tabelle Kasus-Fragen an.

Es wird eine Erfolgsmeldung angezeigt: „Kasus-Frage wurde gelöscht“.

 

 

Fehlerquelle:
Fehler L1: Ohne Auswahl löschen
