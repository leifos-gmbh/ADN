WSD anlegen

Vorbedingungen:

•      Das ADN-System zeigt Tabelle WSDen an.

 

Gehen Sie folgendermaßen vor:

1.  Klicken Sie auf den Aktionsknopf „WSD hinzufügen“.

2.  Das ADN-System zeigt das Formular WSD an.

Im Formular werden Pflichteingaben mit einem Stern „*“ gekennzeichnet.

•      Name               (Text, Länge 100, Pflichtfeld)

•      Behördenkennziffer (Text, Länge 10, Pflichtfeld)

•      Telefon Zentrale           (Text, Länge 30, Pflichtfeld, wird für Einladungen benötigt)

•      Telefax Zentrale           (Text, Länge 30, Pflichtfeld, wird für Einladungen benötigt)

•      E-Mail Zentrale   (Text, Länge 50, Pflichtfeld, wird für Einladungen benötigt)

•      Internet            (Text, Länge 50, Pflichtfeld, wird für Einladungen benötigt)

•      Straße

•      Hausnummer

•      PLZ

•      Ort

•      Separate Besucheradresse (Auswahlbox, falls ausgewählt erscheinen die vier vorangegangen Felder nochmals separat)

•      Bankinstitut                 (Pflichtfelder)

o  BLZ

o  Kontonr.

o  IBAN

o  BIC

•      Kostengegenstände       (alles Pflichtfelder)

o  Bescheinigung

o  Ersatzausfertigung

o  Verlängerung

o  Prüfungsteilnahme

•      jeweils mit den Attributen:

o  Laufende Nummer   (Text, Länge 4)

o  Text/Beschreibung    (Text, Länge 1000)

o  Kosten                   (Numerisch,  zwei Nachkommastellen)

 

Am Ende der Seite befinden sich die Aktionsknöpfe „Speichern“ und „Abbrechen“.

3.  Tragen Sie die Daten korrekt und vollständig ein.

4.  Klicken Sie auf „Speichern“.

5.  Das ADN-System überprüft die Eingaben. Wenn alle Eingaben gültig sind, werden die Daten gespeichert. Es wird die Erfolgsmeldung: „WSD wurde angelegt.“ angezeigt.

6.  Das ADN-System zeigt die Tabelle WSDen an.

 

 

Fehlerquelle:
Fehler A1 Mit leerem Pflichtfeld abspeichern
