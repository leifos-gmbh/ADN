Eingabe

Fehler E1       Eingabe einer für das Prüfungsziel existierenden Fragennummer

Vorfall:

Nach einer Eingabe erfolgt ein Fehlerhinweis

Gehen Sie folgendermaßen vor:

1.  Tragen Sie korrekte Daten ein.

2.  Klicken Sie auf „Speichern“.

 

Fehler E2       Dadurch Über-/Unterschreitung der Gesamtsumme

Vorfall:

Sie haben die Gesamtsumme der Fragen über- oder unterschritten. Das ADN-System zeigt Ihnen nach dem Speichern eine Fehlermeldung an.

Gehen Sie folgendermaßen vor:

1.  Überprüfen Sie die Gesamtsumme.

2.  Tragen Sie korrekte Zahlen ein oder passen Sie die Fragenanzahl je Ziel/Unterziel an.

3.  Klicken Sie auf „Speichern“
