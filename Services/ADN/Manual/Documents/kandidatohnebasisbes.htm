Kandidat ohne Basisbescheinigung für Aufbauprüfung anmelden

 

Hinweis:
Soll ein Prüfungskandidat für eine Aufbauprüfung abspeichert werden, der zwar keine Basisbescheinigung vorweisen kann, aber aus anderen Gründen trotzdem teilnehmen kann/soll, so kann dies angegeben werden, indem Sie eine entsprechende Warnmeldung übergehen.

 

Vorbedingungen:

•      Das ADN-System zeigt das Formular Prüfungskandidaten/Sachkundige an.

 

Gehen Sie folgendermaßen vor:

1.  Wählen Sie aus der Auswahlliste „Zu absolvierende Prüfung“ eine der Optionen „Aufbau Gas“ oder „Aufbau Chemie“ aus.

2.  Markieren Sie die Auswahlboxen „Antrag auf Prüfung gestellt“, aber nicht „Basisbescheinigung aus dem Ausland vorhandenen“.

3.  Klicken Sie auf „Speichern“.

4.  Das ADN-System gibt eine Warnmeldung „Der Benutzer benötigt eine gültige Basisbescheinigung, um an einer Aufbauprüfung teilnehmen zu wollen. Möchten Sie die Daten trotzdem speichern?“ und zwei Aktionslinks „Ja“ und „Nein“ aus.

5.  Klicken Sie auf „Ja“.

6.  Das ADN-System zeigt die Tabelle Prüfungskandidaten/Sachkundige an.

Der Prüfungskandidat taucht in der Liste auf.

In der Spalte „Zu absolvierende Prüfung“ wird die gewählte Aufbauprüfung angezeigt.

Es wird eine Erfolgsmeldung angezeigt: „Prüfungskandidat wurde erstellt.“
