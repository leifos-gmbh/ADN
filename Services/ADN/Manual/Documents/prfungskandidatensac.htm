Prüfungskandidaten/Sachkundigen bearbeiten

Vorbedingungen:

•      Das ADN-System zeigt die Tabelle Prüfungskandidaten/Sachkundige an.

•      Es besteht mindestens ein Prüfungskandidat.

 

Gehen Sie folgendermaßen vor:

1.  Klicken Sie auf den Aktionslink „Kandidat bearbeiten“ zu einem Eintrag.

2.  Das ADN-System zeigt das Formular Prüfungskandidaten/Sachkundige an:

3.  Ändern Sie die Daten so, dass sie korrekt und vollständig bleiben.

4.  Klicken Sie auf „Speichern“.

5.  Das ADN-System zeigt die Tabelle Prüfungskandidaten/Sachkundige an.

Es wird eine Erfolgsmeldung angezeigt: „Prüfungskandidat wurde gespeichert“

 

 

Fehlerquelle:
Fehler A1 Mit leerem Pflichtfeld abspeichern

 

 

 

Hinweis:
Die eindeutige Identifikation eines Kandidaten erfolgt über die Kombination Name / Vorname / Geburtsdatum. Sollten Sie eine Kombination dieser Daten angeben, die im System bereits vorhanden ist (egal, ob Prüfungskandidat, Interessent oder Sachkundiger) gibt das ADN-System eine Sicherheitsabfrage aus: „Ein Kandidat mit diesem Namen und Geburtsdatum ist im System. Wollen Sie trotzdem speichern?“
Bestätigen Sie in diesem Fall, wenn Sie tatsächlich einen Datensatz für eine neue Person anlegen möchten. Es wird ein neuer Datensatz vom System generiert.
