Kandidat mit externer Basisbescheinigung für Aufbauprüfung anmelden

 

Hinweis:
Soll ein Prüfungskandidat für eine Aufbauprüfung angemeldet werden, der zwar im System keine Basisbescheinigung erhalten hat, aber eine Basisbescheinigung aus dem Ausland vorweisen kann, so kann dies durch eine spezielle Option angegeben werden.

 

Vorbedingungen:

•      Das ADN-System zeigt das Formular Prüfungskandidaten/Sachkundige an.

 

Gehen Sie folgendermaßen vor:

1.  Wählen Sie aus der Auswahlliste „Zu absolvierende Prüfung“ eine der Optionen „Aufbau Gas“ oder „Aufbau Chemie“ aus.

2.  Markieren Sie die Auswahlboxen „Antrag auf Prüfung gestellt“ und „Basisbescheinigung aus dem Ausland vorhandenen“.

3.  Klicken Sie auf „Speichern“.

4.  Das ADN-System zeigt die Tabelle Prüfungskandidaten/Sachkundige an.

Der Prüfungskandidat taucht in der Liste auf.

In der Spalte „Zu absolvierende Prüfung“ wird die gewählte Aufbauprüfung angezeigt. Das ADN-System zeigt die Erfolgsmeldung: „Prüfungskandidat wurde erstellt“ an.
