Eingabe einer Sperrfrist

Vorbedingungen:

•      Das ADN-System zeigt das Formular Korrektur Basis oder das Formular Korrektur Aufbau an.

 

Gehen Sie folgendermaßen vor:

1.  Markieren Sie die Auswahlbox Sperrfrist.

2.  Tragen Sie das Datum des Sperrfristendes ein und wählen Sie aus der Auswahlliste der WSDen, durch wen die Sperrfrist festgelegt wurde.

3.  Klicken Sie auf die Schaltfläche „Speichern“.

Das ADN-System speichert die eingetragenen Werte.

Das ADN-System kehrt zurück zur Tabelle Gemeldete Kandidaten. Die Erfolgsmeldung: „Korrektur wurde gespeichert“ wird angezeigt.

 

 

Hinweis:
Eine Sperrfrist darf nur eingetragen werden, wenn eine Prüfung nicht bestanden wurde.
Sollten Sie aus Versehen eine Sperrfrist aktivieren, obwohl die Prüfung bestanden wurde, erscheint eine Warnmeldung: “Es wurde eine Sperrfrist angegeben, obwohl das Ergebnis "bestanden" ist. Bitte korrigieren Sie die Eingabe“. Die eingetragenen Daten werden nicht gespeichert.
