Maximum von einer Frage je Ziel einstellen

 

Hinweis:
Die Vorschriften des ADN sehen in bestimmten Prüfungsbögen vor, dass eine bestimmte Anzahl von Fragen aus mehreren Prüfungszielen gezogen werden, allerdings unter der Maßgabe, dass je Prüfungsziel nur eine Frage ausgewählt wird. Damit soll verhindert werden, dass alle Fragen per Zufall aus einem Ziel gezogen werden.

 

Beispiel Trockengüterschifffahrt

Vorbedingungen:

•      Das ADN-System zeigt das Formular Eintrag Anzahl zu generierender Fragen im Register Trockengüterschifffahrt an.

 

Gehen Sie folgendermaßen vor:

1.  Klicken Sie auf „Eintrag hinzufügen“.

2.   ADN zeigt das Formular Eintrag Anzahl zu generierender Fragen im Register Trockengüterschifffahrt an.

3.  Wählen Sie die gewünschten Prüfungsziele bzw. Unterziele aus.

4.  Markieren Sie die Auswahlbox „Max. eine Frage je Ziel“.

5.  Klicken Sie auf „Speichern“.

6.  Das ADN-System zeigt die Tabelle Anzahl zu generierender Fragen im Register Trockengüterschifffahrt an. Es erscheint die Meldung: „Eintrag wurde hinzugefügt.“. Je nach gewähltem Wert erscheint die entsprechende Information: „Es sind momentan [n] Fragen(n) zu wenig zugeordnet“ bzw. „Es sind momentan [n] Fragen(n) zu viel zugeordnet“.
