Wartungsmodus

Gehen Sie folgendermaßen vor:

1.  Wählen Sie den Menüpunkt „Administration | Wartungsmodus“ aus.

2.  Das ADN-System zeigt das Formular Wartungsmodus an:

Abbildung 107: Formular Wartungsmodus

Das Formular besitzt nur eine Auswahlbox „Wartungsmodus aktiv“.

3.  Markieren Sie die Auswahlbox.

4.  Klicken Sie auf „Übernehmen“.

5.  Das ADN-System überprüft die Eingaben. Wenn alle Eingaben gültig sind, werden die Daten gespeichert. Das ADN-System meldet im Erfolgsfall „Die Einstellungen wurden gespeichert.“

6.  Das ADN-System zeigt weiterhin das Formular Wartungsmodus.

 

 

Hinweis:
Es können im Wartungsmodus keine Prüfbögen generiert werden. Der Menüpunkt Prüfungsvorbereitung/Prüfungsbögen steht nicht zur Verfügung.
