Übersicht der Teilnehmer eines Prüfungstermins für die Ausstellung von ADN-Bescheinigungen aufrufen

Vorbedingungen:

•      Das ADN-System zeigt die Tabelle Prüfungstermine/ADN-Bescheinigungen.

•      Mindestens ein Teilnehmer hat die Prüfung bestanden.

 

 

Hinweis:
In dieser Tabelle werden nur Prüfungskandidaten angezeigt, die die Prüfung bestanden haben.

 

Gehen Sie folgendermaßen vor:

1.  Klicken Sie auf „Prüfungsteilnehmer“ eines Prüfungstermins.

2.  Das ADN-System zeigt die Tabelle Prüfungskandidaten/ADN-Bescheinigungen.

Abbildung 74: Tabelle Prüfungskandidaten/ADN-Bescheinigungen

Über und unter der Tabelle befindet sich der Aktionsknopf „Bescheinigung herunterladen“.

In jeder Zeile wird eine Auswahlbox zur Auswahl der Teilnehmer angezeigt. Diese wird erstmalig angezeigt, wenn eine Bescheinigung erstellt wurde.

Das System erlaubt eine alphabetische bzw. numerische Sortierung des Tabelleninhaltes jeder dafür vorgesehenen Spalte in aufsteigender oder absteigender Reihenfolge.

Die Tabelle besitzt vier Inhaltsspalten mit den Eigenschaften des Prüfungskandidaten:

•      Name

•      Vorname

•      Geburtsdatum

•      Bescheinigung ausgestellt am

 

Je Prüfungsteilnehmer werden folgende Aktionslinks angeboten:

•      Bescheinigung erstellen (wenn noch keine Bescheinigung zur Prüfung erstellt wurde)

•      Bescheinigung bearbeiten (wenn eine Bescheinigung zur Prüfung erstellt wurde)
