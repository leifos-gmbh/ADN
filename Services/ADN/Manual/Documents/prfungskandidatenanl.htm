Prüfungskandidaten anlegen

Vorbedingungen:

•      Das ADN-System zeigt die Tabelle Prüfungskandidaten/Sachkundige an.

Gehen Sie folgendermaßen vor:

1.  Klicken Sie auf den Aktionsknopf „Kandidat hinzufügen“.

2.  Das ADN-System zeigt das Formular Prüfungskandidaten/Sachkundige:

Im Formular werden Pflichteingaben mit einem Stern „*“ gekennzeichnet. Das ADN-System zeigt die folgenden Eingabefelder:

•      Anrede (Auswahlliste „Frau“/“Herr“, Voreinstellung „Herr“, Pflichtfeld)

•      Name (Text, Länge 50, Pflichtfeld)

•      Vorname (Text, Länge 50, Pflichtfeld)

•      Geburtsdatum (Datumsfeld, Pflichtfeld)

•      Staatsangehörigkeit (Auswahlliste aller in der Administration gepflegten Länder, Pflichtfeld)

•      Zu absolvierende Prüfung (Auswahlliste Keine, Trockengüterschifffahrt, Tankschifffahrt, Trockengüter-/Tankschifffahrt kombiniert, Gas, Chemie)

•      Auswahlbox „Antrag auf Prüfung gestellt“

•      Auswahlbox „Basisbescheinigung aus dem Ausland vorhandenen“

•      Telefon (Text, Länge 30)

•      E-Mail (Text, Länge 50)

•      Erfasst durch (Auswahlliste der WSDen aus den Stammdaten, Voreinstellung leer, Pflichtfeld)

•      Bemerkung (Text, CLOB)

•      Sperrfrist (Auswahlbox)

•      Sperrfrist festgelegt durch (Auswahlliste aller WSDen)

•      Sperrfrist bis zum (Datum)

•      Land(Auswahlliste der in den Stammdaten gepflegten Länderkennzeichen)

•      Postleitzahl (Text, Länge 10)

•      Ort (Text, Länge 50)

•      Straße (Text, Länge 50)

•      Hausnummer (Text, Länge 10)

•      Optionale Angaben zur Zustelladresse:

•      Anrede (Auswahlliste „Frau“/“Herr“ (Voreinstellung)

•      Name (Text, Länge 50)

•      Vorname  (Text, Länge 50)

•      Land (Auswahlliste der in den Stammdaten gepflegten Länderkennzeichen)

•      Postleitzahl (Text, Länge 10)

•      Ort (Text, Länge 50)

•      Straße (Text, Länge 50)

•      Hausnummer (Text, Länge 10)

•      Zustelladresse aktiviert (Auswahlbox, Voreinstellung: deaktiviert)

Am Ende der Seite befinden sich die Aktionsknöpfe „Speichern“, „Abbrechen“ und „Speichern und letzte Schulung ändern“.

3.  Tragen Sie die Daten korrekt und vollständig ein.

4.  Klicken Sie auf „Speichern“.

5.  Das ADN-System zeigt die Tabelle Prüfungskandidaten/Sachkundige an.

Der Prüfungskandidat taucht in der Liste auf.

Es wird eine Erfolgsmeldung angezeigt: „Prüfungskandidat wurde erstellt“.

 

 

Fehlerquelle:
Fehler A1 Mit leerem Pflichtfeld abspeichern

 

 

 

Hinweis:
Die eindeutige Identifikation eines Kandidaten erfolgt über die Kombination Name / Vorname / Geburtsdatum. Sollten Sie eine Kombination dieser Daten angeben, die im System bereits vorhanden ist (egal, ob Prüfungskandidat, Interessent oder Sachkundiger) gibt das ADN-System eine Sicherheitsabfrage aus: „Ein Kandidat mit diesem Namen und Geburtsdatum ist im System. Wollen Sie trotzdem speichern?“
Bestätigen Sie in diesem Fall, wenn Sie tatsächlich einen Datensatz für eine neue Person anlegen möchten. Es wird ein neuer Datensatz vom System generiert.
